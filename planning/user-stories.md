- User Stories

- Student

As a student, I want to register an account so that I can access the clinic booking system.

As a student, I want to log in to the system so that I can manage my appointments.

As a student, I want to view available appointments so that I can choose a suitable time.

As a student, I want to book an appointment so that I can schedule a clinic visit.

As a student, I want to cancel an appointment so that I can change my schedule if necessary.

As a student, I want to view my past sessions so that I can keep track of my clinic visits.

As a student, I want to rate a session so that I can give feedback on the service.

- Doctor

As a doctor, I want to access the dashboard so that I can view my scheduled sessions.

As a doctor, I want to view all sessions so that I can see upcoming appointments.

As a doctor, I want to view completed sessions so that I can review past consultations.

As a doctor, I want to view patient details so that I can prepare for consultations.

- Administrator

As an administrator, I want to manage doctor sessions so that the clinic schedule is organized.

As an administrator, I want to view student appointments so that I can monitor clinic bookings.

As an administrator, I want to update or cancel appointments so that the schedule remains accurate.
