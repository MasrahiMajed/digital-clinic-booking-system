-Digital Clinic Booking System

-SDG
SDG 3 – Good Health and Well-Being

-Project Description
The Digital Clinic Booking System helps students book clinic appointments online. Many students wait in long queues when visiting the university clinic. This system allows students to book, manage, and track their appointments easily. The goal is to reduce waiting time and improve clinic service efficiency.
