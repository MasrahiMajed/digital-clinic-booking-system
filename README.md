# Digital Clinic Booking System

## SDG Chosen
SDG 3 – Good Health and Well-Being

## Project Description
The Digital Clinic Booking System is designed to improve and simplify student access to the university clinic. Many students experience long waiting queues when visiting the clinic, which results in wasted time. This system allows students to book, manage, and track appointments online. The goal is to reduce waiting time and improve the overall efficiency of clinic services for students.

## Main Features
- Student registration and login
- View available appointments
- Book appointments
- Cancel appointments
- View sessions
- Rate sessions
- Doctor dashboard
- Admin session management

## SDLC Overview
This project was developed using Agile and Scrum practices. Trello was used to manage product backlog, sprint backlog, and weekly task progress. GitHub was used to organize project files and track development progress. The project includes planning documents, design diagrams, testing notes, and final prototype screens.
